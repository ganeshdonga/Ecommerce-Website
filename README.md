UNA – Simple E-Commerce Shopping Website
UNA is a simple and user-friendly e-commerce website designed to provide an easy online shopping experience. The project includes essential shopping features such as product browsing, category sections, responsive design, and a clean user interface. It aims to demonstrate front-end web development skills using modern web technologies while creating an attractive and functional shopping platform.

Features:
Responsive and clean UI design
Product listing and browsing
Category-based shopping sections
Easy navigation for users
Beginner-friendly e-commerce layout
Technologies Used:
HTML
CSS
JavaScript

This project was developed for learning and improving web development skills by building a basic online shopping platform.
